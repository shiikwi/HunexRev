## Hunex Engine Tool
tools for unpack `Dramatic Create's` game.  
Only tested `Hoshizora Tetsudou to Shiro no Tabi(NS)`

## hpbUnpack
unpack `*.hpb` files.  
Usage: Keep `*.hpb` file and its corresponding `*.hph` file in the same folder.  
```bash
hpbUnpack.exe <hpb_File_Path>
```

## ScriptParser
Unpack `*.heslnk` file, Export `*.hdlg` file
```bash
ScriptParser.exe <-h/-d> <heslnk/hdlg_file_Path>
```
-h: Unpack `*.heslnk` file  
-d: Export `*.hdlg` to .txt
